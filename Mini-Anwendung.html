JavaScript Klassen: Mini-Anwendung "Einfacher Zähler"

Diese Präsentation zeigt, wie Klassen in JavaScript verwendet werden, um eine Mini-Anwendung zu erstellen. Im Fokus steht der Zähler, der durch Kapselung von Daten und Funktionalität das grundlegende Prinzip der objektorientierten Programmierung veranschaulicht.

1. Aufbau der Mini-Anwendung "Einfacher Zähler"

Die Mini-Anwendung besteht aus:

  • Klasse Zaehler: Modelliert das Konzept eines Zählers.
  • Konstruktor (constructor): Initialisiert den Zählerwert mit einem optionalen Startwert (Standard: 0).
  • Methoden:
    • hochzaehlen() – Erhöht den Zählerwert.
    • runterzaehlen() – Verringert den Zählerwert, aber nicht unter 0.
    • anzeigen() – Gibt den aktuellen Zählerwert in der Konsole aus.
    • reset() – Setzt den Zähler auf den Startwert zurück.
    • getWert() – Getter-Methode zur sicheren Abfrage des aktuellen Werts.
  • Instanziierung: const meinZaehler = new Zaehler();

2. Codebeispiel

Das folgende Codebeispiel zeigt, wie die Klasse implementiert und verwendet wird:


class Zaehler {
  constructor(startwert = 0) {
    this.wert = startwert;
  }

  hochzaehlen() {
    this.wert++;
  }

  runterzaehlen() {
    if (this.wert > 0) this.wert--;
  }

  anzeigen() {
    console.log('Aktueller Zählerwert:', this.wert);
  }

  reset() {
    this.wert = 0;
  }

  getWert() {
    return this.wert;
  }
}

// Instanz der Klasse erstellen
const meinZaehler = new Zaehler();

// Methoden aufrufen
meinZaehler.hochzaehlen();
meinZaehler.anzeigen();  // Ausgabe: Aktueller Zählerwert: 1

meinZaehler.runterzaehlen();
meinZaehler.anzeigen();  // Ausgabe: Aktueller Zählerwert: 0

meinZaehler.reset();
meinZaehler.anzeigen();  // Ausgabe: Aktueller Zählerwert: 0

4. Erklärung der Kernkonzepte

Kapselung: Die Klasse Zaehler fasst alle relevanten Daten und Operationen zusammen.

Getter-Methode: getWert() sorgt für eine sichere Abfrage des aktuellen Wertes.

Fehlermanagement: runterzaehlen() verhindert einen negativen Wert.

Modularität: Die Klasse kann einfach in anderen Programmen wiederverwendet werden.
