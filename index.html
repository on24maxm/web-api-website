Kurzanleitung für Javascript Klassen und Klassen-Syntax

Was sind Javascript Klassen?

- Klassen sind eine relativ neue Funktion in Javascript, diese wurden 2015 eingeführt

- Sie sind "synaktischer Zucker" über der prototypenbasierten Vererbung, das bedeutet, dass sie im Grunde eine vereinfachte und intuitivere Schreibweise für das bereits existierende prototypenbasierte Objektmodell von Javascript darstellen.

- Sie ändern die Funktionsweise nicht, machen die objektorientierte Programmierung aber zugänglicher und ähneln der Syntax anderer objektorientierter Sprachen.

Warum sollte man Javascript Klassen nutzen?

- Bessere Stukturierung:
Klassen helfen, verwandten Code (Daten und Funktionen) in logischen Einheiten zu organisieren, was die Lesbarkeit und Wartbarkeit verbessert
- Wiederverwendbarkeit:
Einmal definierte Klassen können beliebig oft verwendet werden, um neue Objekte mit ähnlichen Eigenschaften und Verhaltensweisen zu erstellen
- Einfachere Vererbung:
Die extends-Syntax macht die Vererbung von Eigenschaften und Methoden zwischen Klassen deutlich einfacher und verständlicher als die direkte Manipulation von Prototypen

Schlüsselwörter

/*
            class: Definiert eine neue Klasse.
            constructor(): Spezielle Methode zur Objekt-Initialisierung.
            extends: Ermöglicht Vererbung von einer anderen Klasse.
            super(): Ruft den Konstruktor der Elternklasse auf.
            Methoden: Funktionen innerhalb der Klasse, definieren das Verhalten.
            get/set: Steuern den Zugriff auf Eigenschaften.
            static: Methoden/Eigenschaften, die zur Klasse selbst gehören.
*/
